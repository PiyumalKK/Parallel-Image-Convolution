# Parallel-Image-Convolution

This repository contains code for parallel image convolution using various approaches: Serial, OpenMP, POSIX Threads, MPI, CUDA, and Hybrid methods.

## Structure
- src/: Source code for all implementations
- include/: Header files
- images/: Input and output images
- results/: Benchmark results
- scripts/: Utility scripts

## Usage
See README for compilation and run instructions for each implementation.
